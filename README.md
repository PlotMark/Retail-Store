# Quick start (local)
1. Start MongoDB locally (default URI `mongodb://localhost:27017/retailcrm`)
2. Backend:
   - cd backend
   - npm install
   - npm run dev	# starts server on http://localhost:5000
3. Frontend:
   - cd frontend
   - npm install
   - npm start		# starts React app on http://localhost:3000
